Mastermind

Mastermind is a code-breaking game for two players. One player (the codemaker) creates a secret code, and the other player (the codebreaker) must guess the code in a set number of turns.

How to Play:

  • The codemaker secretly selects a 4-color code using a combination of 6 different colors. The colors can repeat.
  • The codebreaker has 10-12 chances to guess the correct color sequence.
  • After each guess, the codemaker provides feedback:
    • A red peg indicates a correct color in the correct position.
    • A white peg indicates a correct color, but in the wrong position.
    • No peg means the guessed color is not part of the code at all.
  • The codebreaker continues guessing until they either:
    • Crack the code by getting all four colors in the correct positions.
    • Run out of guesses without solving the code, in which case the game ends, and the codemaker wins.
    • After the game ends, the option to restart the game is available to try again with a new secret code.
