Mastermind

This mastermind version is a code-breaking game for one player. The computer will create a secret code, and the player must guess the code in a set number of turns.

  • The computer will select a 4-color code using a combination of 6 different colors. The colors can repeat.
  • The player has 12 chances to guess the correct color sequence.
  • After each guess, the computer provides feedback:
    • A red peg indicates a correct color in the correct position.
    • A white peg indicates a correct color, but in the wrong position.
    • No peg means the guessed color is not part of the code at all.
  • The player continues guessing until they either:
    • Crack the code by getting all four colors in the correct positions.
    • Run out of guesses without solving the code, in which case the game ends, and the computer wins.
    • After the game ends, the option to restart the game is available to try again with a new secret code.
